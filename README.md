# Nikon Framer v2.0 📸

**Nikon Framer v2.0** 是一款专为摄影师打造的现代化、专业级本地相框水印工具。它不仅是一个工具，更是对摄影作品“最后一步”的极致追求。

结合 **Rust (Rayon)** 的巅峰性能与 **Vue 3** 的丝滑交互，v2.0 实现了从“功能可用”到“工业级生产力”的质变。无论是海量样片批处理，还是单张作品的精雕细琢，它都能在守护 100% 本地隐私的前提下，为您带来毫秒级的极速响应。

---

## 🔥 v2.0 重大更新 (Major Updates)

### 💎 极致视觉：现代深度美学 (New Visual Identity)
全新的 **"Depth & Layers"** 设计语言，为专业摄影创作量身打造。
*   **多维空间感**: 采用三段式深度背景（Resource, Preset, Workspace），通过微弱的色阶与阴影，打造出具有空间感的沉浸式工作区。
*   **双模式进化**: 
    *   **深空灰 (Dark Mode)**: 专为后期修图优化的低干扰环境，高对比度呈现画面细节。
    *   **极简白 (Light Mode)**: 模拟画廊展示环境，通过强化的边框与细腻的质感阴影，营造高级的“触觉”体验。
*   **细节至上**: 针对高分辨率屏幕优化的点阵背景（Dot Grid）与动态磨砂玻璃（Glassmorphism）交互。

### 🚀 算力跃迁：多核并行管线 (High-Performance Core)
彻底重构底层图像引擎，将您的 CPU 核心潜力发挥到极致。
*   **Rayon 并行驱动**: 自动调度全核心计算资源，实现真正的全线程负载均衡。
*   **毫秒级批处理**: 相比传统串行处理，大批量导出速度提升高达 **1000%**。
*   **智能任务编排**: 结合原子计数器 (`AtomicUsize`) 与安全终止协议 (`AtomicBool`)，确保处理流程快而不乱，随启随停。

## ✨ 核心功能 (Core Features)

### 🎨 影廊级模版系统
内置多种精心调校的模版，覆盖从纪实到艺术的全场景需求：
*   **经典白底 (Classic White)**: 纯正画廊风格，自动解析并排版 EXIF 信息（相机、镜头、参数）。
*   **氛围模糊 (Atmosphere)**: 智能采样画面色彩，生成具有扩散感的毛玻璃背景，提升作品格调。
*   **宝丽来 (Polaroid)**: 复古即时成像质感，致敬经典美学。
*   **个性签名 (Signature)**: 深度定制化，支持导入个人签名，打造独一无二的品牌标识。

### ⚙️ 专业导出引擎
*   **多格式支持**: 完美支持 JPG (可调质量)、PNG (无损) 与 WebP 导出。
*   **自定义路径**: 灵活选择“原图同级”或“自定义目录”输出，保持工作流整洁。
*   **智能重名检查**: 自动过滤已处理文件，拒绝重复劳动。

### 🛡️ 隐私与安全
*   **100% 离线运行**: 所有图像处理均在本地完成，数据绝不联网，保护版权与隐私。
*   **轻量化架构**: 采用 Tauri 框架，极小的安装包体积，极低的内存占用。

---

## 🛠️ 技术架构 (Technology Stack)

### 前端 (User Interface)
*   **框架**: Vue 3 (Composition API) + Vite
*   **状态**: 响应式 Store 模式，实现复杂场景下的状态共享与联动。
*   **样式**: 现代 CSS 变量体系，支持毫秒级的全局主题无感切换。

### 后端 (Rust Core)
*   **框架**: Tauri v2.0
*   **图像处理**: `image` (编解码) + `rayon` (并行)
*   **元数据**: `kamadak-exif` (专业级 EXIF 解析)
*   **日志系统**: 完整的 `tauri-plugin-log` 集成，随时追踪性能瓶颈。

## 🚀 快速开始 (Getting Started)

### 1. 环境准备
*   安装 [Node.js](https://nodejs.org/) (推荐 LTS)
*   安装 [Rust](https://www.rust-lang.org/tools/install) 环境

### 2. 运行与构建
```bash
# 安装依赖
npm install

# 启动开发预览 (Debug 模式处理速度有限)
npm run tauri dev

# 构建生产版本 (Release 模式全速运行)
npm run tauri build
```
